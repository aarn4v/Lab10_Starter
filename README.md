## Features Implemented
- A/B Testing: 50/50 split for blue vs white background
- Google Analytics 4 integration for event tracking
- Google Tag Manager setup for custom event triggers
- Feature request board for user feedback

## How to Use
1. Open `index.html` in a browser
2. The page will randomly show either a white or blue background (50/50)
3. Stay on the page for 30+ seconds - an event will fire to Google Analytics
4. Analytics data is tracked in real-time at your GA4 dashboard
